About Jett

Jett is a modern deployment service built with a robust microservice architecture. Its frontend is developed using Angular 20 with server-side rendering, paired with an Express backend that leverages Passport for secure authentication. The platform orchestrates deployment workflows through three dedicated microservices: a request handler service to manage user requests, an upload service that integrates with Cloudflare R2 for scalable storage, and a deploy service powered by a GitHub Actions worker to automate project deployments. The request handler and upload services are containerized using Docker and deployed on Render, ensuring reliable scalability and isolation.

To provide real-time feedback, Jett uses a Redis hashset to track the status of deployed projects, allowing users to monitor progress instantly. Beyond personal deployments, the platform also showcases community projects, fostering collaboration and discovery. The seamless integration of these components enables users to manage, deploy, and explore projects efficiently, all while maintaining high performance, security, and scalability. Jett combines modern frontend technologies, cloud-based storage, CI/CD automation, and real-time status tracking into a cohesive deployment ecosystem.
